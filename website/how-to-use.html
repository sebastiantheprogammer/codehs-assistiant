How to Install and Use CodeHS Assistant

Follow these simple steps to get your CodeHS Assistant extension up and running in your browser.

Installation Instructions (for Chrome/Edge)

  1. Download the .zip file: If you haven't already, download the extension .zip file (this link will be provided after purchase).
  2. Unzip the file: Locate the downloaded .zip file and extract its contents to a folder on your computer. Remember the location of this folder.
  3. Open Extension Management:
    • For **Chrome**: Open Chrome, type `chrome://extensions` in the address bar, and press Enter.
    • For **Microsoft Edge**: Open Edge, type `edge://extensions` in the address bar, and press Enter.
  4. Enable Developer Mode: On the Extensions page, toggle on the "Developer mode" switch, usually found in the top right corner.
  5. Load Unpacked: Click the "Load unpacked" button that appears.
  6. Select Extension Folder: Navigate to and select the folder where you unzipped the extension files.
  7. You're Done! The CodeHS Assistant extension should now appear in your browser.

How to Use

Once installed, the CodeHS Assistant extension will appear in your browser's toolbar. Click on the extension icon when you are on a CodeHS assignment page to activate its features. Follow the on-screen prompts to extract the assignment, solve it, and auto-paste the solution.
