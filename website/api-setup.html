How to Set Up Your API Key

  1. Go to OpenAI API Keys and sign in to your account
  2. Click "Create new secret key"
  3. Give your key a name (e.g., "CodeHS Assistant")
  4. Copy the generated API key
  5. Open the CodeHS Assistant extension
  6. Paste your API key into the "API Key" field
  7. Enter your activation code in the "Activation Code" field
  8. Click "Activate" to start your subscription

Important Notes

Your subscription timer will start as soon as you activate the extension with your code.

The timer cannot be paused or reset, so make sure you're ready to use the extension before activating.

If you need to transfer your subscription to a different browser or device, contact our support team.
